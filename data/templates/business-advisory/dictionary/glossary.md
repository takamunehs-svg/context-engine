# 用語集

| 用語 | 定義 |
|---|---|
| stakeholder_alignment | 経営層・意思決定者の納得度（1–5） |
| operating_clarity | 運用設計・責任分界の明確さ（1–5） |
| field_readiness | 現場の展開準備度（1–5） |
| rollout_risk | 横展開の失敗リスク（1–5、高いほど悪い） |
| governance_first_plan | 責任分界と運用ルールの整備を優先するアクション |
| executive_alignment_plan | 経営層向け意思決定資料を先に整えるアクション |
| rollout_enablement_plan | 現場チャンピオンを立てて小さな成功例を作るアクション |
| standard_followup_plan | 論点整理と次回宿題を絞る通常継続アクション |
| GO/ADJUST/STOP | 週次レビューの進行判定 |
| escalation_check | 担当者変更・緊急対応が必要か判定するプロセス |
