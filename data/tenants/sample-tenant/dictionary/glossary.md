# 用語集

| 用語 | 定義 |
|---|---|
| stakeholder_alignment | 意思決定者の納得度。低いほど社内合意が不足している |
| operating_clarity | 運用設計の明確さ。責任分界・例外処理・更新手順が決まっているか |
| field_readiness | 現場の準備度。担当者・業務フロー・初期ユーザーがそろっているか |
| rollout_risk | 展開リスク。高いほど混乱・形骸化・反発の可能性が高い |
| GO/HOLD/STOP | サイクル末の進行判定 |
| governance_first_plan | 展開前に責任分界・運用ルールを固める支援方針 |
| executive_alignment_plan | 意思決定者向けの社内合意形成を優先する支援方針 |
| rollout_enablement_plan | 小さな展開単位で現場定着へ進める支援方針 |
