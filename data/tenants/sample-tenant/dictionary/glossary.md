# 用語集

| 用語 | 定義 |
|---|---|
| NRS | Numeric Rating Scale。疼痛の主観評価（0-10） |
| RPE | Rating of Perceived Exertion。主観的運動強度（6-20、Borgスケール） |
| 可動域 (ROM) | Range of Motion。関節可動域 |
| GO/ADJUST/STOP | サイクル末の進行判定 |
| 介入ミックス | 疼痛管理・機能改善・全身体力・教育の組合せ |
| medical_referral | 医療機関への連携・紹介 |
